# Bank Loan Analysis Dashboard | Power BI

## Overview

This project is an interactive Power BI dashboard developed to analyze bank loan data and provide insights into loan applications, funded amounts, repayments, borrower profiles, and overall portfolio performance.

The dashboard enables users to monitor key lending metrics, evaluate loan quality, identify regional trends, and explore detailed loan records through interactive visualizations.

> **Disclaimer:**  
> This project was created for learning and portfolio purposes. The dataset has not been independently verified and should not be considered an accurate representation of any real financial institution.

---

## Dataset

- **Source:** Sample Financial Loan Dataset
- **File Format:** CSV
- **Records:** Financial loan application data
- **Tool Used for Visualization:** Power BI Desktop

---

## Dashboard Pages

### 1. Summary Dashboard

Provides a high-level overview of lending performance through Key Performance Indicators (KPIs).

#### KPIs

- Total Loan Applications
- Month-to-Date (MTD) Loan Applications
- Total Funded Amount
- Month-to-Date Funded Amount
- Total Amount Received
- Month-to-Date Amount Received
- Average Interest Rate
- Average Debt-to-Income (DTI)

#### Loan Quality Metrics

**Good Loans**

- Good Loan Percentage
- Good Loan Applications
- Good Loan Funded Amount
- Good Loan Amount Received

**Bad Loans**

- Bad Loan Percentage
- Bad Loan Applications
- Bad Loan Funded Amount
- Bad Loan Amount Received

A Loan Status table provides a breakdown of loan performance across different loan statuses.

---

### 2. Overview Dashboard

Focuses on lending trends and borrower characteristics using interactive visualizations.

#### Visualizations

- Monthly Loan Application Trend
- Regional Analysis by State
- Loan Term Distribution
- Employment Length Analysis
- Loan Purpose Analysis
- Home Ownership Distribution

Each visualization supports analysis of:

- Total Loan Applications
- Total Funded Amount
- Total Amount Received

---

### 3. Details Dashboard

Provides a detailed tabular view of loan-level information, allowing users to explore individual loan records and supporting metrics for deeper analysis.
